About the Project

Flash Flood Detection & Early Warning System using AI/ML

🌊 FloodSense AI

FloodSense AI is an advanced machine learning system designed to predict flood probability using a comprehensive set of 20+ environmental, geographical, infrastructure, socioeconomic, and policy-related parameters. The system uses an ensemble of four state-of-the-art ML models to provide accurate, reliable flood risk assessments.

Trained on 50,000 data samples, the system captures complex non-linear relationships between flood-causing factors and outputs a probability score from 0 to 1, categorized into risk levels from "Very Low" to "Very High".

Python Flask TensorFlow Scikit-learn XGBoost Pandas NumPy Chart.js

📊 Dataset

  • 50,000 training samples
  • 20 primary features
  • 4 derived engineered features
  • Flood probability range: 0.0 – 1.0
  • Features span: Climate, Geography, Infrastructure, Environment, Socioeconomic, Policy

🎯 Key Features

  • Real-time prediction using 4 ML models
  • Ensemble averaging for best accuracy
  • Interactive parameter sliders
  • Risk level categorization
  • Actionable recommendations
  • Model performance analytics
  • Feature importance visualization
